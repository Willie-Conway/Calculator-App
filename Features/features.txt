# Features of Calculator-App

## Basic Operations
- **Addition**: Perform simple addition of two or more numbers.
- **Subtraction**: Subtract one number from another.
- **Multiplication**: Multiply two or more numbers.
- **Division**: Divide one number by another (with error handling for division by zero).

## Advanced Features
- **Percentage Calculations**: Easily calculate percentages.
- **Square and Square Root**: Calculate the square and square root of a number.
- **Exponents**: Raise numbers to a specified power.

## User Interface
- **Responsive Design**: User-friendly interface that adapts to different screen sizes.
- **Dark Mode**: Option to switch between light and dark themes.
- **Keyboard Shortcuts**: Support for keyboard inputs for faster calculations.

## History
- **Calculation History**: Keep track of previous calculations for easy reference.
- **Clear History Option**: Option to clear calculation history as needed.

## Accessibility
- **Voice Input**: Option to input calculations using voice commands.
- **High Contrast Mode**: Enhanced visibility for better accessibility.

## Customization
- **Custom Themes**: Ability to choose from various themes and color schemes.
- **Font Size Adjustment**: Option to adjust font size for better readability.

## Additional Features
- **Scientific Calculator Mode**: Switch to a scientific calculator for advanced mathematical functions.
- **Conversion Tools**: Built-in tools for unit and currency conversions.

## Future Plans
- **Graphing Functionality**: Planned feature for plotting functions and equations.
- **Multi-Language Support**: Plans to add support for multiple languages.

## Feedback
- **User Feedback System**: Built-in feature to gather user feedback for continuous improvement.
